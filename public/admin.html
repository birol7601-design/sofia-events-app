Eventbrite Aggregation

Fetches upcoming Sofia events from the Eventbrite API and adds them to the Pending Review queue.
Events are never auto-approved — you review and approve each one manually.
Re-running the fetch is safe — duplicates are automatically skipped.

Setup notes

Requires EVENTBRITE_TOKEN set in Railway environment variables.
If search is blocked (some account tiers), organizer-specific fetching will be needed — ask Birol to provide organizer IDs.
After fetching, go to ⏳ Pending to review and approve events.
